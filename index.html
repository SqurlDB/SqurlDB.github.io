SQURL

What is SQURL?

A cross-platform desktop application designed to generate meaningful dummy data and monitor database KPIs

Features

Generate Meaningful Dummy Data

Create dummy data from a wide variety of categories, applicable to many different sectors

Download Ready-to-Import SQL Files

Download your dummy data as sql files, that can be easily imported into your database.

Create 50MM+ Rows of Data at a Time

Create up to 50MM+ rows of data at any given time, giving you the ability to run database metrics dependent on scale.

Database Dashboard with Key KPIs

Connect to your database and view key KPIs via the built in dashboard tab

Available across all major OS

Usable on Mac, Windows, and Linux based machines
